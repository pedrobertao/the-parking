/parking

GET
Return all plates/car registered
POST
Register a plate by sending a plate like { plate : 'BRA1234' }

/parking/:id/out

PUT
Mark plate/car as leaving the parking lot

/parking/:id/pay

PUT
Mark the car/plate as paid, and need to \n send an amount in the body like { amount : 20 }
